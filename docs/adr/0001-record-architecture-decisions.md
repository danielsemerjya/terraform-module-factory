# 0001 — Record architecture decisions

Date: 2026-06-12. Status: accepted.

## Decision

Record significant decisions about this repo and its harness as ADRs in `docs/adr/`,
numbered sequentially. Agents read ADRs before "improving" something deliberate.

## Context

Agentic development amplifies a failure mode: an agent (or a new human) sees an
unusual-looking choice, assumes it's an accident, and "fixes" it. ADRs make
intentionality machine-readable.
